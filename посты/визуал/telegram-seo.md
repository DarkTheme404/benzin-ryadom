### Описание бота (BotFather → /setdescription)
```
Бензин рядом — бесплатный бот для поиска АЗС с топливом.

Показывает наличие, цены и очереди на 27 000+ АЗС по всей России.
Обновление данных каждый час из 50+ источников.

1. Нажми /start
2. Отправь геолокацию или напиши город

Источники: fuelprice.ru, 2ГИС, 28 TG-каналов, отчёты водителей.

Ищи в любом чате: @benzyn_ryadom_bot 92 Иваново

Команды:
/find — найти АЗС рядом
/subscribe — подписка на push о завозе
```

## 3. Inline-режим — ключ к вирусному росту

### Как это работает
Пользователь пишет в любом чате:
```
@benzyn_ryadom_bot 92 Иваново
```
И видит результат — может отправить другу. Это бесплатный трафик.

### Текст подсказки (уже есть в handlers.py)
Убедись что `INLINE_HINT` содержит ключевые слова:
```
💡 Попробуй: @benzyn_ryadom_bot 92 Иваново или @benzyn_ryadom_bot Лукойл
```

### Что добавить в inline-результаты
Каждый результат должен содержать:
- Название АЗС
- Адрес
- Цену (если есть)
- Статус (✅⚠️❌)

## 7. Бот в каталогах Telegram

### Где разместить:
1. **tgstat.ru** — каталог ботов, статистика
2. **botlist.ru** — каталог Telegram ботов
3. **t.me/BotNews** — официальный канал Telegram о ботах
4. **@StoreBot** — каталог ботов внутри Telegram

### Как добавить:
- Зайти на сайт
- Нажать «Добавить бота»
- Вставить @benzyn_ryadom_bot
- Заполнить описание с ключевыми словами

## 8. Mini App (WebApp)

Если есть Mini App — это ещё +1 точка касания. Убедись что:
- URL добавлен в BotFather (/setmenubutton)
- В Mini App есть SEO-текст на главной

## 9. Периодический контент

### Автопосты в канал @benzyn_ryadom:
- Раз в день: «Где есть бензин в [город]»
- Цены на выходные
- Акции и скидки на АЗС

### Зачем:
- Каждый пост = упоминание бота
- Подписчики канала = потенциальные пользователи бота
- Канал индексируется Telegram-поиском

## 10. Метрики для отслеживания

После внедрения отслеживать:
- `/start` в день (рост = хороший знак)
- Inline-запросов в день (вирусный потенциал)
- Новых подписчиков канала
- Позицию в Telegram-поиске по запросу «бензин»
