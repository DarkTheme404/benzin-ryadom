## 4. Сети АЗС (приоритет 0.75) 🟡
- **Источник**: Официальные виджеты/приложения
  - Лукойл (~2450 АЗС)
  - Газпромнефть (~1657 АЗС)
  - Роснефть (~945 АЗС)
  - Татнефть (~525 АЗС)
  - Башнефть (~часть Роснефти)
- **Статус**: 🟡 Готов скрипт `scripts/parse_networks.py` (старая версия)

### Проблема
У большинства сетей **нет публичных API** — цены только в личном кабинете.
Альтернативы:
- Парсить мобильные приложения через mitmproxy (нарушает ToS)
- Платить за официальный API (нет публичных)
- Использовать сторонние агрегаторы (fuelprice.ru ✅)

## Оркестратор

`scripts/orchestrator.py` — запускает все парсеры.

```bash
# Однократный запуск всех источников
python scripts/orchestrator.py --once

# По расписанию (каждые 6 часов)
python scripts/orchestrator.py --schedule
```

## Cron на Render

Render Free поддерживает **Cron Jobs** (вместо Web Service).

```yaml
# render.yaml
services:
  - type: cron
    name: benzin-parser
    schedule: "0 */6 * * *"  # каждые 6 часов
    runtime: python
    buildCommand: pip install -r requirements.txt
    startCommand: python scripts/orchestrator.py --once
```

## Обновления (Refresh)

Каждый источник имеет интервал:
- **fuelprice.ru**: 24 часа (раз в сутки)
- **Сети АЗС**: 6 часов
- **TG-каналы**: 2 часа (когда будет auth)
- **benzin-price.ru**: 24 часа (когда будет headless)
