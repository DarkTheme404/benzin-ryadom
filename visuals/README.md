# Визуал для постов

## Как сделать скриншот:

1. Открой HTML-файл в браузере
2. Нажми `Cmd + Shift + 4` (Mac) или `Win + Shift + S` (Windows)
3. Выдели область с визуалом
4. Сохрани как PNG

## Файлы:

- `post1.html` — VK пост (1200×628, горизонтальный)
- `post1_tg.html` — TG пост (600×600, квадратный)

## VK размер:
- Пост: 1200×628
- Обложка сообщества: 1640×856
- Аватар: 200×200

## TG размер:
- Пост: 600×600 или 1280×720
- Шапка канала: 640×640
