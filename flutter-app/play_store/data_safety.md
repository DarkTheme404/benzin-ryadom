# Data Safety Declaration — Google Play Store

## Объявление о безопасности данных

### Сбор данных

| Тип данных | Сбор | Использование | Передача |
|---|---|---|---|
| Местоположение | Да | Для поиска ближайших АЗС | Нет |
| Имя | Да | Для профиля пользователя | Нет |
| Город | Да | Для релевантных результатов | Нет |
| Идентификатор устройства | Да | Для аналитики | Нет |

### Практики в области безопасности

- Шифрование данных при передаче
- Возможность удаления данных по запросу
- Данные не продаются третьим лицам

### Политика в отношении конфиденциальности

Политика конфиденциальности доступна по ссылке: [Политика конфиденциальности](privacy_policy.md)

## Data Safety Declaration (English)

### Data Collection

| Data Type | Collected | Used For | Shared |
|---|---|---|---|
| Location | Yes | Finding nearby gas stations | No |
| Name | Yes | User profile | No |
| City | Yes | Relevant results | No |
| Device ID | Yes | Analytics | No |

### Security Practices

- Data encryption during transmission
- Data deletion available upon request
- Data is not sold to third parties
