# Brand Kit — Бензин рядом

## App Identity

- **Name:** Бензин рядом
- **Tagline:** «Где есть бензин прямо сейчас»
- **Category:** Мобильное приложение / Навигация / Авто

---

## Colors

| Role      | Hex       | Usage                        |
| --------- | --------- | ---------------------------- |
| Background | `#0a0e1a` | Main app background, dark UI |
| Accent     | `#ff6b35` | Primary CTA, highlights, links |
| Premium    | `#fbbf24` | Premium badges, stars, upgrades |
| Success    | `#22c55e` | Confirmed reports, green status |

---

## Typography

- **Font family:** Inter
- **Weights:** 400 (body), 500 (emphasis), 600 (subheaders), 700 (headers), 800 (hero titles)
- **Scale:** follow Material Design 3 type scale

---

## Tone of Voice

**Principles:**
- Friendly and informal — use «ты»
- Short sentences, no fluff
- Informative first, witty second
- Subtle humor — never forced
- Action-oriented — guide the user

**We are NOT:**
- Corporate or formal
- Overly enthusiastic or spammy
- Passive-aggressive

---

## Voice Examples

| Empty search result      | «По вашему запросу ничего не найдено» | «В этом районе пока пусто. Попробуй.expand зону поиска» |

---

## Logo Usage

- Minimum clear space: 1.5x logo height on all sides
- On dark backgrounds: use full-color version
- On light backgrounds: use dark variant
- Never stretch, rotate, or recolor the logo

---

## Do's and Don'ts

| ✅ Do                              | ❌ Don't                              |
| ---------------------------------- | ------------------------------------ |
| Keep text short and scannable      | Write long paragraphs               |
| Use accent color for CTAs only     | Paint entire screens accent color    |
| Pair icons with labels             | Use icons without text               |
| Show real data in mockups          | Use Lorem ipsum                      |
| Respect the color ratios           | Override palette freely              |
